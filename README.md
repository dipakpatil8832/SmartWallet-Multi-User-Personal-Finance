# 💳 Personal Wallet & Expense Tracker (Python)

A menu-driven Python application that helps users track daily expenses and income with **date & time support**, maintain a **wallet balance like a bank passbook**, and generate **monthly financial reports**.

---

## 🚀 Features

- ➕ Add money to wallet (income)
- 💸 Record expenses with **custom date & time**
- 💰 Auto update remaining balance
- 📒 View passbook (transaction history)
- 📊 Monthly expense & income report
- 🧾 Category-wise expense breakdown
- 🧑 User-friendly CLI (Command Line Interface)

---

## 🛠 Technologies Used

- **Python 3**
- **OOP (Object-Oriented Programming)**
- `datetime` module
- CLI-based user input
